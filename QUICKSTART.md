# 🚀 快速开始指南

**当前 Git 维护版本：v1.5.8**
**运行时版本：v1.5.8**

## 安装与启动

### 源码运行

```bash
pip3 install -r requirements.txt
python3 jav_file_organizer.py
```

### macOS 直接测试

桌面正式包：

```text
JAVFileOrganizer-v1.5.8.app
```

## 首次使用 JAVLibrary

1. 选择数据源：`JAVLibrary`
2. 点击 `测试连接`
3. 程序会弹出 Chrome
4. 在浏览器中完成 Cloudflare 验证
5. 验证通过后，再次测试连接或直接开始处理

## 推荐测试番号

- **JavHoo / JavBus**：`SONE-753`
- **JAVLibrary**：`SSIS-001`

## Cookie / Profile

JAVLibrary 相关持久化目录：

- Cookie：`~/.jav_organizer/javlibrary_selenium_cookies.pkl`
- Profile：`~/.jav_organizer/javlibrary_chrome_profile`
